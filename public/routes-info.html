GET /product
Returns all products from database. Supports sorting and pagination. You can sort by price, name etc and also fetch page wise data.
Example:
/product
/product?sort=price
/product?sort=-price&page=1&limitdata=3
Open
GET /test
Testing route used for direct filtering using query parameters. By default results are sorted by price (high to low).
Example:
/test?company=apple
/test?rating=4.8
Open
GET /filtered
Advanced filtering route. Supports name search, company filter, sorting and selecting specific fields only.
Example:
/filtered?company=apple
/filtered?name=iph
/filtered?sort=-price
/filtered?select=name,price
Open
POST /add-product
Inserts new product into database using form data. Requires product name, price and company.
Example Body:
